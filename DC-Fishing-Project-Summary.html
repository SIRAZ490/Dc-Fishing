DC Fishing Store

Full Project Documentation — Built with React + Vite + Tailwind CSS

What is This Project?

DC Fishing is a professional e-commerce website for a fishing and diving gear store based in Lebanon. It is a front-end only project, meaning there is no real server or database — all product data is stored locally in a JavaScript file.

The website includes a homepage, shop page with filters, product detail pages, a contact page, and an about us page.

Step-by-Step Project Setup

Step 1 — Create the Project

Run this command in your terminal to create a new React project using Vite:

npm create vite@latest "dc-fishing" -- --template react

This creates a folder called dc-fishing with all the basic React files already set up.

Step 2 — Install Base Dependencies

cd dc-fishing
npm install

This downloads all the default packages that Vite + React need to run.

Step 3 — Install All Project Libraries

npm install tailwindcss @tailwindcss/vite framer-motion swiper zustand react-router-dom lucide-react react-icons react-helmet-async

This installs every library the project uses — animations, icons, routing, cart state, slider, and styling.

Step 4 — Configure Tailwind CSS

Edit the file vite.config.js to add the Tailwind plugin:

import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})

Step 5 — Build the Folder Structure

src/
├── assets/images/       ← Local images saved here
├── components/
│   ├── layout/          ← Navbar, Footer, CartDrawer, AnnouncementBar, ScrollToTop
│   └── home/            ← HeroSlider, CategorySection, FeaturedProducts,
│                           BrandsSection, OffersSection, TestimonialsSection
├── data/
│   └── products.js      ← All product data (name, price, image, category, brand...)
├── pages/               ← HomePage, ShopPage, ProductPage, ContactPage, AboutPage
└── store/
    └── cartStore.js     ← Shopping cart logic (add, remove, update quantity)

Step 6 — Create Product Data

All products are defined in src/data/products.js as a JavaScript array. Each product has: id, name, brand, price, oldPrice, image, category, rating, reviews.

Pages Built

1. HomePage

  • Hero Slider — 3 auto-sliding images (Spearfishing, Fishing, Freediving) with Ken Burns zoom effect
  • Category Section — 3 clickable image cards (Fishing, Spearfishing, Diving)
  • Featured Products — grid of top products with Add to Cart
  • Brands Section — logos of carried brands
  • Offers Section — promotional banners
  • Testimonials Section — customer reviews

2. ShopPage

  • Full product grid — 12 products per page with pagination
  • Filter sidebar with: Category, Price Range (slider), Brand (checkboxes), Min Rating
  • Sort by: Default, Price Low→High, Price High→Low, Top Rated, Name A–Z
  • Search tag display with clear option
  • Clear all filters button

3. ProductPage

  • Single product detail view
  • Image, name, brand, price, rating, description
  • Add to Cart button with quantity selector

4. ContactPage

  • Hero section with background image and overlay
  • Contact form — Name, Email, Subject, Message fields with Send button
  • Contact info cards — Call Us, Email Us, Visit Us (links to Google Maps), Working Hours
  • WhatsApp instant chat button
  • Social media links — Facebook, Instagram, YouTube, TikTok
  • Google Maps embed showing store location

5. AboutPage

  • Hero image with dark overlay and animated title
  • Who We Are — company story text
  • Stats — 10,000+ customers, 500+ products, 15+ brands, 48h delivery
  • Our Values — Premium Quality, Passion First, Expert Support cards
  • Ready to Gear Up — CTA with Shop Now and Contact Us buttons

Layout Components

Navbar

  • Logo, navigation links, search bar, user icon, cart icon with item count badge

CartDrawer

Footer

Key Features

Responsive DesignTailwind CSS breakpoints — works on mobile (sm), tablet (md), and desktop (lg/xl)

AnimationsFramer Motion — fade/slide on page load and scroll-triggered animations
Google MapsEmbedded iframe in Contact page showing store location in Dbayeh, Lebanon

How to Run the Project

cd dc-fishing
npm install        ← install all dependencies (only needed once)
npm run dev        ← start the development server

Then open your browser and go to: http://localhost:5173

Note: This is a front-end only project. There is no real payment system, login system, or database. All data is stored in JavaScript files inside the src/data/ folder.
